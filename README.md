# CSL-стиль для Zotero: ДСТУ 8302:2015 🇺🇦🇬🇧
Це сучасна реалізація бібліографічного стандарту ДСТУ 8302:2015 для Zotero, розроблена з урахуванням потреб науковців, що працюють із двомовними джерелами (українська та англійська).
✨ Особливості стилю
•	Мультимовність: Автоматичне перемикання термінів (с./p., №/no., та ін./et al., дата звернення/accessed).
•	Розумна логіка мови: Якщо поле Language у Zotero порожнє — джерело вважається українським. Якщо вказано будь-яке значення (напр. en) — англійським.
•	Коректна пунктуація: Крапки замість ком у періодиці, вивід Місто : Видавець для книг та звітів.
•	Обробка 4+ авторів: Автоматичне скорочення списку авторів до 3-х осіб з додаванням відповідного мовного маркера (та ін. або et al.).
•	Підтримка DOI та URL: Сучасне оформлення посилань на електронні ресурси.

📥 Як встановити
1.	Скачайте файл стилю dstu-8302-2015 by YNA(PUET) v10.8.csl з цього репозиторію.
   📥 [Завантажити файл стилю (.csl)]([https://github.com](https://github.com/Yuriy-Nabasov/zotero-dstu-8302-2015/dstu-8302-2015 by YNA(PUET) v10.8.csl)
3.	Відкрийте Zotero.
4.	Перейдіть у меню:
o	Windows: Правка → Налаштування → Цитування → Стилі.
o	Mac: Zotero → Settings → Cite → Styles.
5.	Натисніть кнопку + (плюс) під списком стилів.
6.	Виберіть завантажений файл .csl та підтвердіть встановлення.
________________________________________
⚙️ Налаштування карток у Zotero (Важливо!)
Для ідеального результату заповнюйте поля в Zotero наступним чином:
1. Перемикач мови
•	Українське джерело: Залиште поле Language (Мова) порожнім.
•	Англійське джерело: Напишіть у полі Language будь-яке значення (наприклад, en або англійська).
2. Типи документів
•	Звіти (Report): Заповнюйте поле Institution (Заклад) для назви організації та Place (Місто).
•	Книги (Book): Заповнюйте Publisher (Видавець) та Place (Місто).
•	Веб-сторінки (Webpage): Вказуйте Website Title (Назва сайту). Місто та Видавець для цього типу ігноруються за вимогами ДСТУ.
3. Автори-організації
Якщо автором є організація (напр. Європейська Бізнес Асоціація):
•	Перемкніть поле автора в Zotero в режим «Одне поле» (Single Field).
•	В кінці назви організації рекомендується поставити крапку (.), щоб відокремити її від назви документа.
4. Сторінки
•	Для Книг/Звітів: Вказуйте загальну кількість (напр. 74). Стиль сам додасть 74 с. або 74 p..
•	Для Статей: Вказуйте діапазон (напр. 10-15). Стиль додасть С. 10-15.
________________________________________
🚀 Поради щодо використання
•	Використовуйте Ctrl+Shift+C, щоб миттєво скопіювати оформлене джерело в буфер обміну.
•	У налаштуваннях Zotero (Експорт) виберіть цей стиль як «Формат за замовчуванням», щоб прискорити роботу.
